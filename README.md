# atcoder

## install

- nvm

  [nvm](https://github.com/nvm-sh/nvm)

- node.js
  ```bash
  $ nvm install --lts
  ```

- npm packages
  ```bash
  $ npm install -g typescript typesync
  ```

- clone this repository

## set up

```bash
$ cd $your_git_repository
$ npm install
```

## run

```bash
$ npm run start
```
